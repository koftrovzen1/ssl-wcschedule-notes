# ssl-wcschedule-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态网页形式存放，不依赖于任何特定域名或后端服务，旨在提供一种轻量、可维护的信息记录与展示方式。

## 目录结构

```
ssl-wcschedule-notes/
│
├── README.md          # 本文件
├── index.html         # 仓库首页 / 导航页（可选）
└── pages/             # 存放所有独立 HTML 页面
    ├── page-1.html
    ├── page-2.html
    └── ...
```

- `pages/` 目录下的每个 HTML 文件均为一个独立的归档页面，可单独访问或通过 `index.html` 进行导航。

## 页面归档说明

- 每个 HTML 文件均为自包含的静态页面，无需后端支持。
- 页面内容以数据展示、记录或说明性信息为主，不涉及动态交互或外部数据源。
- 页面命名建议使用简短、有意义的英文或拼音，便于识别和管理。

## 维护说明

- 新增页面时，请将 HTML 文件放入 `pages/` 目录，并更新 `index.html` 中的导航链接（如有）。
- 请保持页面结构简洁，避免引入过多外部资源，以维持仓库的轻量和可移植性。
- 如对页面内容进行修改，建议在提交信息中简要说明变更内容。
- 本仓库不托管商业或营销内容，仅用于个人或团队内部归档用途。

## 许可

本仓库内容仅供归档参考，不提供任何形式的担保或技术支持。如有需要，可自行 Fork 或复制使用。
